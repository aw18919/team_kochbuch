# 📖 Team Kochbuch

Herzlich willkommen beim **Team Kochbuch** – unserem gemeinsamen Git-Übungsrepository!

---

## 🎯 Zweck

Dieses Repository dient als Übungsplatz, um den Umgang mit Git und GitHub im Team zu erlernen und zu festigen. Hier sammeln wir gemeinsam unsere Lieblingsrezepte und üben dabei typische Git-Workflows.

---

## 🚀 Git-Konzepte die wir üben

- **Branches erstellen** und zwischen ihnen wechseln
- **Commits** sinnvoll schreiben und strukturieren
- **Merge-Konflikte** lösen
---

---

## 🛠️ Erste Schritte

1. Repository klonen:
   ```bash
   git clone https://github.com/dein-name/team_kochbuch.git
   ```
3. Änderungen committen:
   ```bash
   git add .
   git commit -m "Rezept: Mein Lieblingsrezept hinzugefügt"
   ```
4. Branch pushen und Pull Request erstellen:
   ```bash
   git push origin rezept/mein-lieblingsrezept
   ```
